    1. What is Semantic HTML? 

A system for marking up webpages in order to give them meaning to the browser and to developers.
    
    2. What is HTML used for?

    3. What is an attribute and where do we put it? 

An attribute goes inside an HTML tag and effects the tag in different ways such as causing an anchor tag to visit a URL.

An attribute that allows us to make an html element unique from other elements with the same tag.
